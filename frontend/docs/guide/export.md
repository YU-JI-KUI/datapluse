# 数据导出

标注完成后，进入「数据导出」页面，将高质量标注数据下载到本地，用于模型训练或其他下游任务。

## 推荐导出状态

| 状态 | 质量 | 适用场景 |
|------|------|---------|
| `checked` ✅ | 最高 | 经过冲突检测，标注一致性有保障，**优先选择** |
| `annotated` | 较高 | 已完成人工标注，未经冲突检测 |
| `pre_annotated` | 中等 | 仅模型预标注，未经人工确认 |

建议优先导出 `checked` 状态的数据用于训练，这批数据经过多重质量校验。

## 导出格式

支持以下三种格式：

- **Excel**（`.xlsx`）：适合人工查阅和统计分析
- **CSV**（`.csv`）：通用格式，适合直接导入训练框架
- **JSON**（`.json`）：结构化数据，适合程序化处理

## 自定义导出字段

可以自由勾选需要包含的字段，常用字段说明：

| 字段 | 说明 |
|------|------|
| `content` | 文本内容（必选） |
| `label` | 最终意图标签 |
| `label_source` | 标签来源：`auto`（多数投票）/ `manual`（人工裁决） |
| `model_pred` | 模型预测标签 |
| `model_score` | 模型置信度（0～1） |
| `annotator` | 标注员用户名 |
| `annotator_count` | 参与标注人数 |
| `conflict_flag` | 是否存在冲突 |
| `conflict_type` | 冲突类型（label / semantic） |
| `created_at` | 数据入库时间 |
| `annotated_at` | 标注完成时间 |

::: tip 最简训练格式
只需要 `content` 和 `label` 两个字段即可构成最基本的意图识别训练集。
:::

## 导出操作

1. 选择要导出的数据状态（支持多选）
2. 勾选需要的字段
3. 选择导出格式
4. 点击「导出」，浏览器自动下载文件

导出过程在服务端完成，大数据集可能需要几秒钟，请耐心等待。

## 数据探查

「数据探查」页面（左侧导航）提供更灵活的数据浏览功能：

- 按状态、意图标签、标注员、时间范围多维筛选
- 查看每条数据的完整历史（所有标注员的记录、冲突详情）
- 支持直接下载当前筛选结果

::: info 数据探查 vs 数据导出
数据探查适合精细化查看和临时筛选；数据导出适合批量下载完整数据集用于训练。
:::
