# 指标口径

本页说明评测报告中每个指标的计算方式，便于理解数字来源、判断异常。

## 评测来源分布

整份日志按处理方式拆为三类，三者之和等于日志总条数：

```
日志总条数 = 活动标问数 + 规则命中数 + 大模型评测数
             (跳过不评)   (免调模型,计指标)  (真正调模型)
```

## 分发准确率

```
分发准确率 = 判对数 ÷ 参与评分数
```

- **判对**：大模型判「是否本 BU 承接」与日志事实「实际是否分给本 BU」一致（该接且接了，或该拒且拒了）。
- **参与评分数**：大模型给出有效分发判定的样本数。模型出错或缺字段的样本不计入分母。
- **两类错误**（占比分母同为参与评分数）：
  - 漏收：判该接，实际没分到本 BU。
  - 误收：判该拒，实际却分到了本 BU。

该指标以大模型判断与日志客观事实比对，不依赖人工金标，因此生产模式也能计算。

## 问题解决率

```
问题解决率 = 已解决数 ÷ 实际分到本 BU 的样本数
```

- **分子**：判定为「已解决」的样本（「部分解决」「未解决」都不计入）。
- **分母**：只算实际分到本 BU 的样本。拒识或没分到本 BU 的不计入分母。

## 需人工复核数

- 大模型自判「需人工复核」的条数，触发场景包括疑似合规风险、信息严重不足、明显自相矛盾。
- 每被人工复核一条，该数减一（无论复核是否改变判定，都视为已人工确认，移出待复核队列）。

## 业务分类分布

按大模型打的业务分类计数。采用闭集约束，报告中只出现配置的分类加「非 XX 业务」兜底桶，不会冒出几十个模型自创的分类。

## 会话轮次分布

按会话统计，非按样本：

- 每通对话的总轮次 = 该会话内最大轮次（一通五轮对话计为一个「五轮会话」）。
- 动态分桶：一轮、两轮、三轮各单独一档，高轮次按中位数动态切分，区间由数据决定，最多五档。

## 校准指标 κ / F1

仅校准模式（Excel 含金标列）计算：

- **κ（Cohen's Kappa）**：衡量大模型判定与人工金标的一致性，扣除了「碰巧一致」的部分。κ = 1 表示完全一致，κ = 0 相当于随机猜测。样本不均衡时比准确率更可信。
- **F1**：精确率与召回率的调和平均，衡量判定的综合准确性。

## 速查

- 分发准确率的分母排除模型出错的样本。
- 问题解决率的分母只算实际分到本 BU 的样本。
- 业务分类永远是闭集：配置的 N 个加「非 XX 业务」。
