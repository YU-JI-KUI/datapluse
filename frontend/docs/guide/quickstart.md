# 快速开始

五步完成从数据入库到标注导出的完整流程。

## 第一步：创建数据集

管理员进入左侧导航 **数据集管理**，点击右上角「新建数据集」，填写名称和可选描述。

创建完成后，点击该数据集行的 **分配用户** 按钮（👥），勾选需要参与标注的标注员账号，保存。

::: info
左上角的数据集切换器用于选择当前工作的数据集。所有功能都基于当前选中的数据集。
:::

## 第二步：上传数据

进入 **数据上传** 页面，将待标注的文本文件拖拽到上传区，或点击选择文件。

支持的格式：
- **Excel**（`.xlsx`）：优先识别列名 `text` / `content` / `文本` / `内容`
- **CSV**：识别规则同 Excel
- **JSON**：支持字符串数组，或含 `content` / `text` 字段的对象数组

上传完成后会提示「新增 N 条，跳过 M 条」。跳过的条目是已存在的重复数据，系统自动去重。

## 第三步：运行 Pipeline

进入 **Dashboard**，点击右上角「运行 Pipeline」。

系统会在后台依次执行三个步骤：
1. **数据清洗** — 规范化文本格式
2. **LLM 预标注** — 自动预测意图标签
3. **冲突检测** — 标记多人标注不一致的条目

Pipeline 在后台运行，运行期间可以正常使用其他页面，不需要等待。完成后 Dashboard 会展示各步骤的结果摘要。

## 第四步：人工标注

Pipeline 完成后，进入 **标注工作台**。

系统会自动展示待标注条目，并附带 LLM 的预测标签和置信度作为参考。标注员点击正确的意图标签即完成提交，自动进入下一条。

::: tip
置信度 ≥ 90% 时预测结果通常可靠，可直接确认；置信度较低时建议仔细核对文本含义。
:::

## 第五步：导出数据

标注工作完成后，进入 **数据导出** 页面，选择需要包含的字段和导出格式，点击「导出」即可下载。
