# 标注工作台

标注工作台是标注员日常使用的核心页面，用于逐条确认或修改 LLM 的预标注结果，形成最终的意图标签。

## 界面说明

进入左侧导航「标注工作台」，界面分为两个区域：

**左侧：待标注队列**

显示当前数据集中等待标注的条目列表，支持按状态筛选（全部 / 未标注 / 已标注）。点击任意条目可切换到对应数据。

**右侧：标注操作区**

- 文本内容展示区
- LLM 预测标签和置信度（橙色徽标，仅供参考）
- 意图标签选择区（点击即提交）
- 评论框（可选，记录标注理由或疑问）

## 标注流程

1. 系统自动加载第一条待标注数据
2. 阅读文本，参考模型预测（若有）
3. 点击正确的意图标签 → 自动保存并跳转到下一条
4. 如有疑问，在评论框填写备注，再提交标注

点击标签即视为确认提交，无需额外点击「保存」按钮。

::: tip 快速标注高置信度数据
当模型置信度 ≥ 90% 时，预测结果通常准确。可以直接点击预测标签（已高亮显示），快速完成确认。
:::

## 撤销标注

对于自己已提交的标注，在「已标注」视图中找到该条目，点击「撤销」即可撤回，数据状态恢复为待标注，可以重新标注。

只能撤销自己提交的标注，无法撤销其他标注员的记录。

## 多人标注

同一条数据可以由多位标注员独立标注，各自的结果会被系统记录。

当多人给出不同标签时，系统会在 Pipeline 的冲突检测步骤中将该条目标记为「标签冲突」，等待标注员裁决。

::: info 关于最终标签
系统以投票方式自动汇总多人标注结果（少数服从多数）。无冲突时自动确认最终标签；有冲突时由管理员手动裁决。最终标签展示在「数据探查」页面的 `label` 字段。
:::

## 导航与筛选

- **状态筛选**：切换「全部」「未标注」「已标注」，方便在大数据集中定位
- **关键词搜索**：在左侧搜索框输入关键词，筛选特定文本内容
- **翻页**：底部分页控件切换页码，支持每页显示 10 / 25 / 50 / 100 条

## 标注进度

Dashboard 页面的进度看板中会展示当前数据集的整体标注进度，包括已标注条数和剩余待标注条数。
