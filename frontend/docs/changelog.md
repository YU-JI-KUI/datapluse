# 更新日志

## v1.1 · 2026-05-08

### 新功能

- **业务分类管理**：新增独立的业务分类（`t_category`）管理页面，支持增删改查、Excel 批量导入，管理与数据集绑定的分类体系
- **COT 结构化字段**：标注时可填写三个结构化字段——业务分类（下拉选）、关键词、关键词说明，记录每条标注的推理依据；字段由配置中心的 `require_cot` 开关控制显示
- **数据管理列新增**：数据管理页新增"业务分类"和"关键词"两列，支持按业务分类过滤

### 优化

- **数据管理**（原 Data Explorer）：页面改名为"数据管理"，操作列由 4 个独立按钮改为 3-dot 下拉菜单，移除"创建时间"列，减少水平滚动
- **导航改名**："Dashboard" → "首页"，"Data Explorer" → "数据管理"
- **日志修复**：上传文件时的 access log 不再打印 xlsx 二进制内容，改为只记录文件名、大小和格式
- **批量删除**：业务分类页支持多选批量删除

## v1.0 · 2026-05-01

首个正式版本，覆盖数据生产全流程。

- **数据上传**：支持 Excel / CSV / JSON，自动文本清洗与去重
- **Pipeline**：一键触发三步自动化（清洗 → LLM 预标注 → 冲突检测），后台异步执行
- **人工标注**：多人协作标注工作台，支持撤销，结果自动多数投票
- **冲突检测**：标签冲突 + 语义冲突双维度检测，支持单条和批量裁决
- **向量化**：Embedding 向量存库，每日凌晨 2 点自动更新，支持手动触发
- **数据导出**：按状态筛选，字段可自选，支持 Excel / CSV / JSON
- **用户管理**：三角色权限模型（admin / annotator / viewer），支持多数据集隔离
- **Dashboard**：实时展示 Pipeline 进度与向量化状态
