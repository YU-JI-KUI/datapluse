# 更新日志

## v1.0 · 2026-05-01

首个正式版本，覆盖数据生产全流程。

- **数据上传**：支持 Excel / CSV / JSON，自动文本清洗与去重
- **Pipeline**：一键触发三步自动化（清洗 → LLM 预标注 → 冲突检测），后台异步执行
- **人工标注**：多人协作标注工作台，支持撤销，结果自动多数投票
- **冲突检测**：标签冲突 + 语义冲突双维度检测，支持单条和批量裁决
- **向量化**：Embedding 向量存库，每日凌晨 2 点自动更新，支持手动触发
- **数据导出**：按状态筛选，字段可自选，支持 Excel / CSV / JSON
- **用户管理**：三角色权限模型（admin / annotator / viewer），支持多数据集隔离
- **Dashboard**：实时展示 Pipeline 进度与向量化状态
