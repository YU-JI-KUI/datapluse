## v1.5 · 2026-07-17

围绕 AI 评测的时间维度分析、多文件评测与规则配置做了一轮增强。

### 新功能

- **多文件合并评测**：一个评测任务可一次上传多个日志文件合并评测。日志被运营平台按 5 万行拆成多个文件时，直接全选上传即可，无需再手动合并；同一通对话跨文件也能自动接上、按提问时间正确归日
- **按提问时间筛选**：问题洞察支持按日志里的真实提问时间选择日期区间，进入页面默认只看有数据的时段，不再一进来就加载全部历史
- **趋势对比**：问题洞察新增趋势对比视图，直观展示解决率、分发准确率随提问日期的变化，并标注区间涨跌
- **每日频率分来源**：每日频率图拆成「日志总量 / 活动标问 / 短路规则 / AI 评测」四条趋势线，一眼看清每天的日志由哪几种方式处理（日志总量 = 三者之和）
- **搜索返回识别**：自动识别「返回搜索/快捷服务入口」这类回答并统一归类，配合规则可将其判为已解决，减少人工核对
- **规则触发问题支持通配**：短路规则的触发问题支持通配与包含匹配——`*` 匹配任意问题、`%注销%` 匹配包含「注销」的问题、`注销%` 匹配以「注销」开头等，一条规则即可覆盖大量相似问法

### 优化

- **用词统一**：「规则短路」统一为「短路规则」，表述更顺口
- **菜单调整**：问题洞察移到评测菜单末尾，与配置类页面分组更清晰

## v1.2 · 2026-07-01

AI 评测模块正式上线，用于评估对话系统的回答质量。

### 评测核心

- **一键评测**：上传对话日志 Excel，系统对每条客户问题调用大模型逐条判定，汇总为完整评测报告
- **核心指标**：分发准确率、问题解决率、需人工复核数、业务分类分布、会话轮次分布，校准模式下额外提供 κ / F1 一致性指标
- **评测报告**：独立报告页，含来源分布图、业务分类分布、会话轮次动态分桶图，指标附口径说明
- **校准 / 生产双模式**：Excel 含金标列时进入校准模式验证判定可信度，无金标时进入生产模式

### 配置与调优

- **业务分类管理**：按 BU 维护业务分类清单，采用闭集约束，清单外分类统一归入兜底桶
- **活动标问**：固定按钮触发的固定回复整条跳过，不计入指标
- **短路规则**：结果确定的问答配置写死判定，命中即免调大模型，仍计入指标
- **提示词管理**：按 BU 分槽位编辑 Judge 提示词，编辑器支持 Markdown 实时预览，评测时快照隔离
- **人工复核**：可覆盖大模型判定，指标按最终值实时增量重算，无需重跑
- **闭环调优**：详情抽屉内编辑业务知识、单条试跑验证、一键保存，明细勾选后台异步重跑

### 稳定性

- **独立后台 worker**：评测在独立进程执行，不影响主站响应
- **多 POD 一致性**：抢占式调度、全局串行、心跳续约、崩溃自动接管续跑
- **断点续跑与限流退避**：任务中断只补未完成行，触发限流自动暂停退避

## v1.0 · 2026-05-01

首个正式版本，覆盖数据生产全流程。

- **数据上传**：支持 Excel / CSV / JSON，自动文本清洗与去重
- **Pipeline**：一键触发三步自动化（清洗 → LLM 预标注 → 冲突检测），后台异步执行
- **人工标注**：多人协作标注工作台，支持撤销，结果自动多数投票
- **冲突检测**：标签冲突 + 语义冲突双维度检测，支持单条和批量裁决
- **向量化**：Embedding 向量存库，每日凌晨 2 点自动更新，支持手动触发
- **数据导出**：按状态筛选，字段可自选，支持 Excel / CSV / JSON
- **用户管理**：三角色权限模型（admin / annotator / viewer），支持多数据集隔离
- **Dashboard**：实时展示 Pipeline 进度与向量化状态
