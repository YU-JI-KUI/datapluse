# 更新日志

记录 Datapulse 平台的功能更新与改进，最新版本在最前面。

---

## v0.8 · 2026-05-01

### 新功能

- **深色模式**：侧边栏新增明暗主题切换按钮（太阳 / 月亮图标），一键切换，偏好持久化保存，刷新后不会闪烁
- **Pipeline 状态重置**：数据集管理页面新增橙色重置按钮（🔄），可直接在界面重置卡住的 Pipeline 状态，无需操作数据库
- **向量化状态独立展示**：Dashboard 将向量化（Embedding）任务从主流程中分离，在右侧状态面板的「离线任务」区域单独展示，主流程清晰呈现三步

### 改进

- Dashboard Pipeline 进度改为三步展示（清洗 → 预标注 → 冲突检测），与实际执行逻辑一致
- 数据集管理页面重置弹窗新增「同时重置向量化状态」选项，灵活控制重置范围

---

## v0.7 · 2026-04-28

### 新功能

- **批量冲突裁决**：冲突检测页面支持多选批量裁决，大量同类冲突可一次性处理
- **撤销自检冲突**：支持批量撤销由系统自动检测的冲突（区别于标注员标签冲突）
- **冲突列表分页**：冲突数据量大时支持后端分页，避免前端渲染卡顿

### 改进

- 冲突检测不再将已通过检测的 `checked` 数据回退为 `annotated`，减少不必要的重复裁决
- 自检逻辑排除已被管理员 manual 裁决的数据，防止反复触发

---

## v0.6 · 2026-04-20

### 新功能

- **向量化存 PostgreSQL**：Embedding 向量改为持久化存储在数据库（`t_embedding` 表），不再依赖 `.npy` 文件，支持数据集删除时联动清理
- **语义冲突检测**：向量化完成后，冲突检测支持通过余弦相似度发现语义高度相似但标签不一致的数据对
- **数据探查批量修改**：数据探查页面支持多选数据批量修改标注标签，适合大规模标注校正

### 改进

- 冲突检测改用批量向量加载（`load_batch`），避免每条数据单独查询数据库，显著提升检测速度
- 数据集删除时自动清理关联的向量数据

---

## v0.5 · 2026-04-10

### 新功能

- **多人标注与冲突检测**：支持多名标注员独立标注同一条数据，Pipeline 自动检测标签冲突并交由管理员裁决
- **标注撤销**：标注员可撤销自己提交的标注记录，数据重新进入待标注队列
- **标注工作台视图切换**：支持按「全部 / 未标注 / 已标注」筛选，方便跟踪个人进度

### 改进

- 标注结果架构分为 `t_annotation`（标注事实）和 `t_annotation_result`（最终结果），职责分离，裁决逻辑更清晰
- 管理员裁决冲突时只更新最终结果，不修改标注员的原始记录

---

## v0.4 · 2026-03-25

### 新功能

- **数据集管理**：支持创建多个独立数据集，按数据集隔离数据、标注记录和 Pipeline 状态
- **用户分配**：管理员可将标注员账号分配到指定数据集，标注员只能看到被分配的数据集
- **数据集切换器**：左侧侧边栏顶部新增数据集选择器，切换后所有页面自动刷新

---

## v0.3 · 2026-03-10

### 新功能

- **Pipeline 自动化**：三步自动化流程（数据清洗 → LLM 预标注 → 冲突检测），一键触发，后台运行不阻塞界面
- **Dashboard 进度看板**：实时展示 Pipeline 运行进度、速度，以及各步骤的结果摘要
- **LLM 预标注**：集成大语言模型，自动为 `cleaned` 状态数据预测意图标签和置信度分数

### 改进

- 数据清洗增加文本规范化：去首尾空白、压缩连续空格换行、删除 ASCII 控制字符

---

## v0.2 · 2026-02-20

### 新功能

- **标注工作台**：标注员逐条确认或修改预标注结果，点击标签即提交
- **数据导出**：支持按状态筛选导出，格式支持 Excel、CSV、JSON，字段可自定义
- **SHA-256 自动去重**：上传时对文本内容做哈希去重，跨文件和文件内均支持

---

## v0.1 · 2026-02-01

### 首次发布

- 基础数据上传：支持 Excel、CSV、JSON 格式，自动识别文本列
- 用户系统：管理员 / 标注员双角色权限模型
- 数据状态流转：`raw` → `cleaned` → `pre_annotated` → `annotated` → `checked`
