# 维度：全局问题解决率诊断

下面是本次评测中，**已分发到本 BU** 的问题里「没有解决」的整体情况。未解决原因被归为四类：
- **答非所问**：答案没答到用户问的点。
- **信息不全**：答到了方向但关键信息缺失。
- **事实存疑**：答案内容有误或与事实冲突。
- **分发错误**：本不该本 BU 接，硬答导致没解决（归分发侧）。

下面给出四类原因的**全局分布计数**和跨业务分类的失败样例（含原始问题、AI 答案、原因归类）。
